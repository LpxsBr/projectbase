# Template de projeto

## Como usar

para usar, clone o projeto usando:

```
git clone https://github.com/LpxsBr/projectbase.git [nome-do-seu-projeto]
```

Depois:

```bash

cd [nome-do-seu-projeto] && code . && npm run dev

```
